## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Install agents directly from the Web UI dashboard. For agents that are not installed, show an 'Install' button on the agent card. Use the default installation method per agent: npm install -g (Claude Code, Codex, Gemini CLI, OpenCode, Pi, Cline), pip install (Aider), or link to download page (Cursor, Windsurf, Zed, Roo Code). Backend: POST /api/agents/:id/install endpoint that resolves the install command from the update provider's source type and runs it. Frontend: Install button on not-installed agent cards, spinner during install, success/error feedback. Re-detect agent after install to update status.
