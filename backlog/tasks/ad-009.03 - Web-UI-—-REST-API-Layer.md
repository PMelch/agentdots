## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
JSON REST API served by the built-in HTTP server. Endpoints: GET /api/agents (list all), GET /api/agents/:id (detail), GET /api/rules?scope=project|global (list rules), POST /api/rules/sync (trigger sync), GET /api/rules/diff/:agentId, GET /api/skills?scope=, POST /api/skills/sync, GET /api/mcp?scope=, POST /api/mcp/sync, GET /api/updates (check updates), POST /api/updates/:id (trigger update), GET /api/config, PUT /api/config. All endpoints call existing manager/registry modules. Router in src/web/api.ts.
